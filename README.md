# 🚀 بات پرامپت‌نویس حرفه‌ای تلگرام

یک بات هوشمند تخصصی برای نوشتن پرامپت‌های سطح بالا برای Grok، GPT-4o، Claude و Gemini.

### ویژگی‌ها
- منوی دکمه‌ای پیشرفته
- پشتیبانی از پیام صوتی
- ذخیره پرامپت‌های مورد علاقه
- انتخاب مدل هوش مصنوعی
- مهندسی پرامپت حرفه‌ای

### نصب محلی
```bash
pip install -r requirements.txt
cp .env.example .env
# توکن‌ها را در .env وارد کنید
python bot.py
```

### دیپلوی روی Railway
1. پروژه را روی GitHub آپلود کنید.
2. به [railway.app](https://railway.app) بروید.
3. New Project → Deploy from GitHub.
4. متغیرهای `TELEGRAM_TOKEN` و `AI_API_KEY` را تنظیم کنید.
